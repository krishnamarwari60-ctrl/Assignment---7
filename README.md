##    Assignment 7   DATA ANALYSIS WITH SQL


    
##  Description

This assignment demonstrates the use of SQL to manage student records and analyze academic results.
Student and score data are stored using separate tables for better organization.
SQL views are created to simplify access to student scores and identify students who passed all subjects.
A passing score of 40 is used to determine successful students.



## Task performed
1. Created tables to store student details and subject-wise scores
2. Inserted sample data into the tables
3. Created a view to display student scores
4. Created a view to show students who passed all subjects
5. Retrieved results using SELECT queries
